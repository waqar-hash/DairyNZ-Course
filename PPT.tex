\begin{frame}{Pedagogy}
\phantomsection\label{pedagogy}
\href{https://orchid00.github.io/The_Carpentries_info/overview-of-carpentries-pedagogic-model.html}{Inspired
by the carpentries}

\begin{itemize}
\tightlist
\item
  Active learning (practice) is key, this course will have minimal
  components where I speak to you i.e.~lecture. Instead, we will utilise
  `live coding'.

  \begin{itemize}
  \tightlist
  \item
    Live coding will show you my workflow, my mistakes (everyone makes
    them) and how I troubleshoot them!
  \item
    When you run into difficulty utilise the chat in class,
    communicating with others is key to understanding
  \item
    Spaced retrieval
  \item
    Managed cognitive load
  \end{itemize}
\end{itemize}
\end{frame}
